# Robotics Fundamentals

This repository documents my implementation-focused study of robotics fundamentals, with emphasis on industrial manipulators, trajectory generation, and robot-assisted inspection.

## Projects

0. Manifolds and Lie groups (implemented)
1. Rigid-body transformations (implemented)
2. Forward kinematics and Jacobians (under implementation)
3. Inverse kinematics (not implemented yet)
4. Trajectory generation (not implemented yet)
5. Scan-path generation (not implemented yet)
6. Feasibility-aware planning (not implemented yet)

## References

The main reference for the theoretical foundations in this repository is:

- Kevin M. Lynch and Frank C. Park, *Modern Robotics: Mechanics, Planning, and Control*, Cambridge University Press, 2017.

The implementations in this repository are intended as study exercises based on selected concepts from the book.
